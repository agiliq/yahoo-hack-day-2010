Overriding idea
~~~~~~~~~~~~~~~~~~~~~~

A lot of photographers use flickr. They would like to create a site where they can create a portfolio of their flickr photos and set a price on them.

Overview
~~~~~~~~~~~

1. Asks user to allow to login to Flickr to authorize access.
2. Gets the users flickr photos.
3. Allows setting the price on the Photos, a default price, or on a per photo.
4. Gets the users paypal address.
5. Uses the Paypal Website Payments standard to create payment links.
